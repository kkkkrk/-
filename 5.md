## Queue (배열 기반)
- FIFO
- Array, Linked list로 구현 가능
- BFS에 사용
- 지정된 크기 이상으로 Insert 하면 Overflow 발생
- 비어있는데 Delete, Peek하면 Underflow 발생
- front, rear(초기값: 0 or -1)
- Circular Buffer로 구현(지정된 공간의 끝까지 할당되면 다시 공간의 처음부터 사용)
- Print Buffer, Hot Potato Game, 기수 정렬, Bank Simulation
#### ✅ 시간복잡도
Insert: O(1)  
Delete: O(1)  
Peek: O(1)  
Print Queue: O(n)  
#### ✅ 구현
```cpp
#include "Queue.h"

void Error(string s) {
    cerr << "[Error] " << s << endl;
    exit(-1);
}

void Queue::Insert(Data d) {
    if (IsFull()) Error("Queue is full");
    rear = (rear + 1) % QUEUE_SIZE;
    data[rear] = d;
    length++;
}

Data Queue::Delete() {
    if (IsEmpty()) Error("Queue is empty");
    Data out = data[front];
    front = (front + 1) % QUEUE_SIZE;
    length--;
    return out;
}

Data Queue::Peek() {
    if (IsEmpty()) Error("Queue is empty");
    return data[front];
}

void Queue::PrintQueue() {
    cout << "Queue Elements: ";
    for (int i = 0; i < length; i++) {
        cout << data[(front + i) % QUEUE_SIZE] << " ";
    }
    cout << endl;
}

void Queue::PrintBuffer() {
    cout << "Buffer State: [ ";
    for (int i = 0; i < QUEUE_SIZE; i++) {
        cout << data[i] << " ";
    }
    cout << "]" << endl;
}
```
#### ✅ Dynamic Array
공간이 부족할 때마다 queueSize를 두배로 늘림
#### ✅ Radix Sort
1. 숫자들을 현재 기준이 되는 자리수의 숫자에 맞춰 0~9번 큐에 집어넣는다
2. 0~9번큐 순서대로 안에 있는 숫자들을 순서대로 안에 있는 숫자들을 모두 꺼내어 원래 배열에 덮어쓴다(queue를 사용하므로 먼저 들어간 것이 먼저 나온다)
3. 최대 자리수까지 위 과정을 반복한다.
#### ✅ Double-Ended Queue
: Deque  
앞뒤로 넣고 빼고 다 가능  
Circular Buffer 사용  
front: 1, rear: 할당된 공간 마지막 주소로 시작  
delete, pop, peek 모두 O(1)  
```cpp
#include "Deque.h"
void Error(string s) { cerr << s; exit(-1); }

Data Deque::PeekRear()
{ // Get rear element without deleting
    if (IsEmpty()) Error("Queue is empty");
    return data[rear];
}

Data Deque::DeleteRear()
{ // Delete rear element
    if (IsEmpty()) Error("Queue is empty");
    Data out = data[rear];
    rear = (rear - 1 + QUEUE_SIZE) % QUEUE_SIZE;
    length--;
    return out;
}

void Deque::InsertFront(Data d)
{ // Insert a new element at the front
    if (IsFull()) Error("Queue is full");
    front = (front - 1 + QUEUE_SIZE) % QUEUE_SIZE;
    data[front] = d;
    length++;
}
```
