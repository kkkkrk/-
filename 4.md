## Stack(Array 기반)
- LIFO
- DFS, Fuctinon Call에서 사용됨
- 예제: 10진수 2진수로 변환, 에디터 뒤로 가기, 괄호체크
- 지정된 크기 이상으로 push하면 overflow 발생함
- 비어있는데 pop, peek하면 underflow 발생
- top: 데이터가 있는 가장 최근에 넣은 값의 인덱스(초기값: -1) or 가장 최근에 넣은 값의 인덱스 + 1(초기값: 0)
#### ✅ 구현
stack.h
```cpp
#include <iostream>
#include <cstdlib>

using namespace std;

const int STACK_SIZE = 100;
typedef int Data;

class Stack {
    Data data[STACK_SIZE];
    int top;

public:
    Stack() { top = -1; }
    
    void Push(Data d);
    Data Pop();
    Data Peek();
    
    bool IsEmpty() { return top == -1; }
    bool IsFull() { return top == STACK_SIZE - 1; }
    
    void PrintStack();
    int Length() { return top + 1; }
};
```
```cpp
#include "Stack.h"

void Error(string s) { 
    cerr << s; 
    exit(-1); 
}

// O(1)
void Stack::Push(Data d) {
    // Insert a new element to stack
    if (IsFull()) Error("Stack is full");
    data[++top] = d;
}

// O(1)
Data Stack::Pop() {
    // Delete latest element from stack
    if (IsEmpty()) Error("Stack is empty");
    return data[top--];
}

// O(1)
Data Stack::Peek() {
    // Get latest element without deleting
    if (IsEmpty()) Error("Stack is empty");
    return data[top];
}

// O(n)
void Stack::PrintStack() {
    // Print stack elements
    cout << "Stack: ";
    for (int i = 0; i <= top; i++) {
        cout << data[i] << " ";
    }
    cout << endl;
}
```
