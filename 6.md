## List
- 추상 자료형
- 중복 가능
- 순서 중요
- Linear data structure
#### ✅ Main Operations
Insert: 0~length  
Delete: 0~length-1  
Peek: 0~length-1  
Is List Empty?  
그외: Search Element with Key, Print List, Is List Full?, Insert Front, Insert Rear, Delete Front, Delete Rear
